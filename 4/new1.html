introduction

javascript is a cross platform, object-oriented scripting language. it is a
and lightweight language. inside a host environment (for example, a w
browser), javascript can be connected to the objects of its environment
provided programming control over them.

javascript contains a standard library of objects,such as Array, Date,
and a core set of language elements such as operators, control structure
statements. core javascript can be extended for a variety of purposes
supplementing it with additional objects; for example:

  • client side javascript extends the core language by supplying
    control a browser and its Documentation Object Model (DOM). For
    client side extensions allow an application to place elements
    HTML form and respond to user events such as mouse clicks
    input, and page navigation.
  • server-side javascript extends the core language by supplying relevant
    to running javascript on a server. for example, server
    extensions provide continuity of information from one invocation to another
    application, or perform file manipulations on a server.
